==깃허브 브랜치==

$ git branch (브랜치생성)
$ git checkout -b 브랜치이름  (브랜치생성 후 생성된 브랜치로 전환)
$ git branch 	(현재 연결된 브랜치 확인)
$ git checkout 브랜치이름 	(브랜치 전환,vsCode사용시엔 왼쪽 아래 버튼) 
$ git push origin 브랜치 이름   (브랜치에 작업파일 push, vsCode사용시엔 소스제어 버튼 이용)


==sw precache 라이브러리 이용==
구글에서 sw precache검색후 나온 사이트들 참조

1. $ npm install --global sw-precache
2. sw-config.js 파일을 만든 뒤  module.exports 명령어들을 복사해 붙여넣기, 파일 경로들 수정
3.  $ sw-precache --config sw-config.js
4. index.html 파일에서 sw.js를 새로 만들어진 service-worker.js로 바꿈
